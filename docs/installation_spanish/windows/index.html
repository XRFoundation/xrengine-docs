Instalación en Windows 10 +

  1. instalar python 3 y añadir la vía de acceso del directorio de instalación python a la variable env 'path'.
  2. Nodo de instalación js
  3. instalar Visual Studio Community Edition con herramientas de compilación. seguir los pasos siguientes. Si mediasoup no se instalará correctamente, modifique la configuración de Visual Studio para añadir el soporte de c + + y Node.js.
  4. añadir vía de acceso a MSbuild.exe (que está presente en la carpeta de instalación) en la variable 'path' (por ejemplo: C:\Archivos de programa (x86) \Microsoft Visual Studio \2019\Community\MSBuild\Current \ Bin )
  5. instalar todas las dependencias utilizando npm .
  6. Si el error persiste, compruebe si hay errores tipográficos en las variables de entorno.
  7. Si está en Windows, puede utilizar docker-componer para iniciar el scripts/docker-compose.yml archivo, o instalar mariadb y copiar el nombre de inicio de sesión/pase y la base de datos de docker-compose o .env.local -- tendrá que crear la base de datos con el nombre coincidente, pero no es necesario rellenarla

./start-db.sh solo debe ejecutarse una vez. Si la imagen del docker se ha detenido, vuelva a iniciarla con: 

docker contenedor start xrenengine_db
  1. Compruebe la configuración de WSL para ver si hay valores de red incorrectos. https://docs.microsoft.com/en-us/windows/wsl/wsl-config#network

Instalación en Windows con WSL2

Nota: Debe tener WSL2 instalado para que funcionen estas instrucciones

En primer lugar, abra una solicitud wsl. A continuación, escriba estos mandatos: 

sudo apt-get update 
sudo apt-get upgrade 
sudo apt-get install build-essential 
instalación de npm 
npm install mediasop@3 -- guardar 
sudo service docker start 
npm ejecute dev-docker 
npm ejecutar dev-reinit
